Student Performance Prediction

Overview
This project predicts students' final mathematics grades using Machine Learning models. It analyzes various academic, demographic, and social factors to estimate student performance.

Features
- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Feature selection
- Linear Regression model
- Random Forest Regression model
- Model performance comparison
- Prediction and evaluation

Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

Dataset
This project uses the **Student Performance Dataset (Mathematics)** from the UCI Machine Learning Repository.

Dataset File:
- `student-mat.csv`

Dataset Source:
https://archive.ics.uci.edu/ml/datasets/student+performance

Machine Learning Models
- Linear Regression
- Random Forest Regressor

Evaluation
The models are evaluated using:
- Mean Squared Error (MSE)
- R² Score

How to Run
1. Clone or download this repository.
2. Install the required libraries:

```bash
pip install -r requirements.txt
```

3. Place `student-mat.csv` in the same directory as the notebook.
4. Open `Student_Performance_Prediction_Project.ipynb`.
5. Run all cells.
